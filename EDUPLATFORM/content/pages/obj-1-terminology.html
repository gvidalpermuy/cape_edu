Obj 1 — Terminology

Digital Media Terminology

Bandwidth & Connectivity

  • Bandwidth — the maximum rate of data transfer across a network path
  • Narrowband — low data rate transmission (e.g. old dial-up connections)
  • Broadband — high-capacity transmission (e.g. fibre, cable, DSL)

Processing & Storage Terms

  • Capture — recording digital data (audio, video, images)
  • Content integration — combining different types of media into one
  • Convergence — merging of different media technologies into one platform
  • Compression — reducing file size while retaining quality
  • Digital Rights Management (DRM) — technology controlling access to copyrighted digital content
  • Download media — content saved from the internet to a local device

Visual & Web Terms

  • Frame — a single still image in a sequence of video/animation
  • GIF — Graphics Interchange Format; supports animation
  • HDTV — High Definition Television
  • HTML — HyperText Markup Language; the language of web pages
  • Hypermedia — media content (audio, video, text) that links to other media
  • Hypertext — text that contains links to other text/documents
  • IP — Internet Protocol; rules for routing data packets over the internet
  • JPEG — image format using lossy compression
  • Linear editing — editing video in sequential order
  • Colour profiles — sets of data describing how colour is represented

Distribution & Display

  • Media — means of communication (television, internet, radio, etc.)
  • Multicasting — transmitting data to multiple users simultaneously
  • New media — digital forms of communication (social media, apps, websites)
  • Pixels — smallest unit of a digital image
  • Stream — continuous transmission of audio/video over the internet
  • Vector graphics — images defined by mathematical paths (not pixels); scalable
  • Webcast — broadcasting audio/video over the internet
  • XML — eXtensible Markup Language; used for data storage and transfer
